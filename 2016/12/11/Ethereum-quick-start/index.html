Ethereum installation

The official-site :Ethereum in github
Installing from PPA:

sudo apt-get install software-properties-common
sudo add-apt-repository -y ppa:ethereum/ethereum
sudo apt-get update
sudo apt-get install ethereum

Download the ethereum-wallet(choose the latest version 0.87) Download link

Build private-chain

Before we run the bootstrap, we need to install expect by
sudo apt-get install expect

Download ethereum-Bootstrap.

git clone https://github.com/cryptape/ethereum-bootstrap.git

  • 进入本仓库目录: cd ethereum-bootstrap
  • 导入测试账户私钥: ./bin/import_keys.sh
  • 初始化blockchain: ./bin/private_blockchain_init.sh
  • 启动私有链节点: ./bin/private_blockchain.sh.启动成功后可以看到类似如下输出:
    此时以太坊交互式控制台已经启动，我们可以开始测试和开发了。

Truffle

The official-site : Truffle

Preparation : need nodejs5.xx or more , python2.7 and solc

  • install latest nodejs
  • install solc npm install solc
  • install the testrpc by npm install -g ethereumjs-testrpc
  • install the Truffle by npm install -g truffle (the -g may require sudo). To verify it installed, type truffle list in a console window to list all truffle commands.
